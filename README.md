# GHAR Foundation - Official Website

Official website for GHAR Foundation, a German-registered humanitarian 
organization providing aid to crisis-affected regions in Sudan and Yemen.

## Tech Stack

- **Frontend:** Next.js 14 (App Router)
- **CMS:** Sanity.io
- **Styling:** Tailwind CSS
- **Database:** Supabase
- **Payments:** PayPal / Stripe
- **i18n:** next-intl (EN / AR / DE)
- **Hosting:** Vercel

## Pages

- Home
- About
- Projects (Sudan - Yemen)
- Donate
- Partners
- Transparency Reports
- Volunteer & Jobs
- Contact

## Development

This project follows Agile methodology with 5 sprints.
